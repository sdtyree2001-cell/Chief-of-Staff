## Threat Assessment

### Tier 1: Direct Competitors (SaaS + Data Custody)

#### **Sorcero** (HIGHEST THREAT)
- **Model:** SaaS platform; customer sends data to Sorcero cloud
- **Customers:** Large pharma (legacy install base)
- **Price:** Premium (~2x Aurivian target)
- **Weakness:** Data custody concerns (post-GDPR, post-healthcare privacy concerns); slow to innovate on GenAI; not truly autonomous agents
- **Our Advantage:** Half the price + data stays in customer environment + AI-native ops
- **Hot Intel Needed:** Sorcero's GenAI roadmap, customer churn risk, pricing pressure

#### **Within3** (Mid-tier threat)
- **Model:** KOL network + intelligence overlay
- **Customers:** Mid-market pharma, some large pharma
- **Strength:** Relationship-based; strong KOL data
- **Weakness:** Not GenAI-native; expensive services layer
- **Hot Intel Needed:** GenAI investment, churn metrics

---

### Tier 2: Platform Infrastructure (Data Environment)

#### **Snowflake Cortex** (HIGHEST STRATEGIC THREAT)
- **Model:** Customer data warehouse + Cortex AI agents (in-environment)
- **Deployment:** On-premise (customer's Snowflake environment)
- **Current User:** Otsuka Pharmaceuticals (your current employer)
- **Advantage:** Data never leaves Otsuka; can run agents on proprietary data
- **Risk to Aurivian:** If large pharma build homegrown agents on Snowflake, they don't need external vendors
- **Our Counter:** Specialized expertise (MA workflows beat generic Cortex agents); speed-to-value; cost

#### **Microsoft Copilot** (STRATEGIC THREAT)
- **Model:** Cloud + enterprise integration (Teams, Office, Azure)
- **Current User:** Otsuka Pharmaceuticals
- **Advantage:** Built into workflow; low friction for enterprises
- **Risk to Aurivian:** Enterprises default to Copilot before trying external vendors
- **Our Counter:** MA-specific intelligence > generic Copilot; external signal + context that Copilot can't provide alone

---

### Tier 3: Consultancies / Agencies (Service-Based)

#### **ZS Associates, IQVIA, Axtria** (Revenue Risk)
- **Model:** Services + staff augmentation (FTEs, project-based)
- **Customers:** Large pharma (custom solutions)
- **Strength:** Relationships, deep domain knowledge, custom delivery
- **Weakness:** Expensive, slow, not scalable
- **Our Counter:** 10x cheaper, 10x faster, 24/7 (humans can't compete)

---

### Tier 4: Internal Building (Strategic Threat)

#### **In-House Agent Development** (GROWING THREAT)
- **Model:** Companies using Snowflake Cortex, Copilot, Claude API to build internal agents
- **Trigger:** Post-GDPR + healthcare privacy concerns making external vendors risky
- **Example:** Otsuka building on Snowflake Cortex (you know this firsthand)
- **Our Counter:** Specialize on MA workflows + speed-to-value beats slow internal builds
